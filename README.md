# 📊 GTCO Financial Performance & Risk Analysis ## 🎯 Project Overview A Power BI dashboard analysing 11 months of GTCO transaction data, revealing a 103% debit-to-credit ratio and identifying key risk patterns across ₦13.98M in total debits. ## 🛠️ Tools Used - **Power BI** — Dashboard design and visualisation - **DAX** — Calculated measures and KPIs - **Power Query** — Data cleaning and transformation ## 📌 Key Findings - Total Debits: ₦13,979,473 vs Credits: ₦13,622,792 - Debit ratio of **103%** — spending exceeds income - Cash withdrawals dominate at **~65%** of all transactions - Mysterious **October activity drop** requiring investigation ## 📸 Dashboard Preview ![Dashboard Screenshot](dashboard-screenshot.png) ## 📂 Files in This Repository | File | Description | |------|-------------| | `gtco-dashboard.pbix` | Power BI source file | | `dashboard-screenshot.png` | Full dashboard image | | `gtco-article.html` | Full written analysis | ## 🔗 Read the Full Analysis [Click here to read the detailed data story](gtco-article.html)
